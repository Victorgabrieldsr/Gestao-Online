# 📊 Gerenciador de Investimentos Online

## Sobre

O **Gerenciador de Investimentos Online** é uma plataforma web criada para ajudar usuários a **organizar e acompanhar seus investimentos em um único lugar**.

O sistema permite registrar e monitorar investimentos em **ações, criptomoedas e fundos imobiliários (FIIs)**. A plataforma utiliza **APIs de mercado** para buscar periodicamente a cotação dos ativos e atualizar automaticamente o valor dos investimentos.

Além do controle de investimentos, o projeto também está evoluindo para incluir **gestão financeira pessoal**, permitindo que o usuário registre **gastos, depósitos e movimentações financeiras**, facilitando o controle do seu dinheiro.

O objetivo do projeto é oferecer um **dashboard simples e centralizado** para acompanhar investimentos e gerenciar finanças pessoais.

---

## Funcionalidades Atuais

* 📈 Cadastro e gerenciamento de **ações**
* 🪙 Controle de **investimentos em criptomoedas**
* 🏢 Registro de **fundos imobiliários (FIIs)**
* 🔄 Atualização automática de **cotações via API**
* 📊 Acompanhamento do valor total dos investimentos
* 💻 Visualização através de **dashboard web**

## Como a Plataforma Funciona

1. O usuário cadastra seus investimentos (cripto, ações ou FIIs).
2. O sistema consulta periodicamente uma **API de mercado** para obter os preços atualizados.
3. A plataforma atualiza automaticamente o valor total dos ativos.
4. O usuário pode acompanhar seu portfólio através de um **dashboard**.
5. Ferramentas financeiras adicionais permitem controlar **gastos, depósitos e saldo financeiro**.
